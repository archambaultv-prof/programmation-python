# Formulaire d'inscription

## 🎯 Objectif pédagogique

- Apprendre à créer une interface utilisateur simple.
- Apprendre à valider les champs d'un formulaire.

## 📜 Énoncé

Écrire une application Tkinter qui permet afficher un formulaire d'inscription. Le formulaire doit contenir les champs suivants :
- **Nom** : minimum 3 caractères
- **Prénom** : minimum 3 caractères
- **Adresse e-mail** : doit contenir un "@" et un ".", être non vide avant et après le "@", non vide avant et après le ".". Vous pouvez assumer le reste de l'adresse e-mail est composé de lettres et de chiffres
- **Mot de passe** : minimum 6 caractères, dont au moins une lettre et un chiffre
- **Confirmation du mot de passe** : identique au mot de passe
- Un bouton "S'inscrire"
- Un bouton "Annuler"

Le bouton "S'inscrire" doit valider les champs du formulaire. Si tous les
champs sont valides, un message de succès doit être affiché. Sinon, un message
d'erreur doit être affiché. Prévoir une étiquette pour afficher le message de
succès ou d'erreur. Le bouton "Annuler" doit réinitialiser le formulaire. Le
mot de passe doit être masqué (c'est-à-dire que les caractères doivent être
remplacés par des astérisques).

# 🆔 Fiche d'identité

## 🎯 Objectif pédagogique

Écrire un script simple avec entrée et sortie.

## 📜 Énoncé

Vous allez écrire un script qui :

- Demande successivement à l’utilisateur :
    - Son prénom
    - Son nom
    - Son âge (en années)
    - Sa ville

- Convertit l’âge en nombre entier (puisque l’utilisateur aura saisi du texte).

- Calcule l’âge de l’utilisateur en jours. Faire une approximation avec 365 jours par an, sans années bissextiles.

- Affiche un récapitulatif, par exemple : 

  ```
  Bonjour, je m'appelle Alice Dupont, j'ai 25 ans et j'habite à Montréal.
  J'ai environ 9125 jours.
  ```

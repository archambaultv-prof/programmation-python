# Carnet d'adresses (JSON)

## 🎯 Objectif pédagogique

- Comprendre la structure d'un fichier JSON
- Savoir lire un fichier JSON avec `json.load`
- Savoir extraire et filtrer des données

## 📜 Énoncé

Vous disposez d’un <a href="/programmation-python/exercises/json_contacts.json" download="contacts.json">fichier JSON contacts.json</a>
 contenant une liste de
contacts, chacun avec un nom, un courriel et un ou plusieurs numéros de
téléphone. Voici un extrait :

```json
[
  {
    "nom": "Alice Tremblay",
    "email": "alice@example.com",
    "telephones": ["000-123-4567", "000-999-0000"]
  },
  {
    "nom": "Bob Dubois",
    "email": "bob@example.com",
    "telephones": ["000-765-4321"]
  }
]
```

Écrire un programme Python qui :

1. Charge le fichier JSON dans une variable.
2. Affiche tous les noms et leurs courriels.
3. Affiche les personnes qui ont **plus d’un numéro de téléphone**.

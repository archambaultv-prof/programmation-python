# Analyse de ventes (CSV)

## 🎯 Objectif pédagogique

- Savoir lire un fichier CSV
- Savoir manipuler les données extraites
- Savoir effectuer des calculs simples (somme, moyenne)

## 📜 Énoncé

On vous fournit un <a href="/programmation-python/exercises/csv_ventes.csv"
download="ventes.csv">fichier CSV ventes.csv</a> contenant les ventes d’un
magasin. Chaque ligne représente une transaction, avec les colonnes suivantes.
Voici un extrait :

```
Date,Produit,Quantité,PrixUnitaire
2024-01-01,Clavier,2,49.99
2024-01-01,Souris,1,24.99
```

Écrire un programme Python qui :

1. Lit le fichier CSV.
2. Affiche le nombre total de transactions.
3. Calcule et affiche le chiffre d'affaires total (Quantité × Prix).
4. Calcule et affiche la valeur moyenne d'une transaction.
5. Affiche le chiffre d'affaires total **par produit**.

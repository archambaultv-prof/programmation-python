# Analyse d'un poème (TXT)

## 🎯 Objectif pédagogique

- Savoir lire un fichier texte
- Savoir manipuler les données extraites

## 📜 Énoncé

On vous fournit un <a href="/programmation-python/exercises/txt_poeme.txt"
download="poeme.txt">fichier texte poeme.txt</a> contenant un poème. Écrire un programme Python qui :

1. Lit le fichier texte.
2. Affiche le nombre total de lignes.
3. Affiche le nombre total de caractères.
4. Affiche le nombre total de mots.
5. Affiche l'auteur du poème.

:::tip

Vous n'avez pas besoin d'écrire des fonctions complexes pour cet exercice.
Soyez astucieux et utilisez les méthodes de base pour manipuler les chaînes de
caractères et les fichiers.

:::
